This page is a collection of examples of triggers that can be used with the Vesta repository tools.

Also, don't miss the examples included in the vesta/repos_ui package.

Verbose List

/PropagateMergeHistory

Copy an attribute recording merge history.

/AutoContinuation

Set the "continued-in" attribute used by vupdate.

/AutoBranchStstuas

Set an attribute denoting a branch is closed when it ghets merged into another version.
