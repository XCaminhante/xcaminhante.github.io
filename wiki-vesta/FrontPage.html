Vesta Wiki

Vesta is an advanced software configuration management and build system. For more detailed information see the Vesta homepage.

You can frequently find people discussing Vesta on the Vesta IRC channel.

The VestaFAQ is available to answer your questions.

Interesting pages you'll find here:

  • Documentation and help for users:

    • LearningVesta: Starting point for learning about vesta

      • See also CoreAspects

    • Installing Vesta on Gentoo Linux

    • std_env: How the standard build environment is put together

    • TroubleShooting: error messages + problems you might encounter, and advice on how to fix / workaround them

  • Tips and tricks:

    • SdlHacks: A collection of code snippets in the Vesta builder's language

    • RepositoryTricks: Unusual ways to use the Vesta repository tools.

    • TriggerExamples: Various examples of how you can use the pre/post triggers that the Vesta repository tools provide.

    • ToolIntegration: Making other tools understand Vesta

  • Discussion of future development:

    • ReleasePlans describes the work to be done for upcoming releases of Vesta

    • There is some discussion about future development of VestaWeb.

    • WishList: Discussion and ideas on things that it would be nice to have, if someone would like to try

    • NewVersionSelection: Discussion of issues with the current version override methods and how we might build something better

    • MergingFuture: Adding automated merging capabilities to Vesta

    • ScalingUp: handling 1000s of clients per server, and large build results shipped as symlinks accessed concurrently from many clients

    • DevelopingVesta has some information for people who want to work on Vesta itself

  • General discussion of other topics:

    • ReorganizingRepos: changing/refactoring appendable and immutable directories in the repository

    • TakingVestaIntoUse: One new user's questions

    • UserImpressions: A place for users to record/discuss their experiences with Vesta

    • ThinkingAboutBigPackages : Efficiently using large packages in Vesta

    • PerforceAndVesta : Using Perforce for revision control and Vesta as the builder.

How to use this site

A Wiki is a collaborative site, anyone can contribute and share:

  • Edit any page by pressing Edit at the top or the bottom of the page. Sadly, due to an increasing amount of automated spam, before you can edit pages you must follow a couple preliminary steps:

    1. Create and account by going to the UserPreferences page

    2. Request that your account be added to AllowedToWriteGroup by sending an e-mail to <wiki-admin AT vestasys DOT org> (ideally from an e-mail address that is already subscribed to the vesta-users mailing list).

    3. Once the administrators grant you write access, log into your account before editing.

  • Create a link to another page with joined capitalized words (like WikiSandBox) or with ["quoted words in brackets"]

  • Search for page titles or text within pages using the search box at the top of any page

  • See HelpForBeginners to get you going, HelpContents for all help pages.
