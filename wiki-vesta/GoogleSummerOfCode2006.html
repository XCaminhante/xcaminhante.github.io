Google is again holding their Summer of Code this year. We're considering becoming a mentoring organization. What projects would we have a student work on (aside from any random thing from the SourceForge trakcer)?

  • Implement an interactive Vesta SDL Debugger

  • Finish the implementation of triggers

  • Windows port of repos_ui

  • autoconf-based Vesta build (ideally including a functional port on one or more new platforms)

  • repos_ui GUI based on ReposUIPackageClass

  • Implement CrudeToolImport

  • Implement NFS v4 in the repository server (see "NFS lock support")

  • Get make_compat to be able to build some significant existing make-based build (e.g. gcc, the Linux kernel)

  • Implement compression of immutable files in the repository

  • Implement secure authentication
