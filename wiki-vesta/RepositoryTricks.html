This page is a collection of unusual or interesting things that you can do with the Vesta repository tools.

Verbose List

/SubdirOldVer

Use a subdirectory of a version as the old version.

/LinkIntoWorkDir

Link an existing version into a working directory.

/SymLinkSubstitute

Using the 2 tricks above, create a "link" to a subdir.

/LinkSessionToNewWork

Link an existing session dir into a new working directory, eg, if you happened to delete your work-dir by accident

/MoveVersions

Scripts to help you migrate existing versions/builds to a new hierarchy

Alphabetical List
