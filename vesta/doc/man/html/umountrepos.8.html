umountrepos(8)

Name

umountrepos - Unmount the Vesta repository

Synopsis

umountrepos

Description

See the Terminology section of the vesta-intro man page for definitions of terms.

The umountrepos command is a simple shell script to unmount the Vesta repository from the local host.

Diagnostics

Here are the most common error messages you are likely to see and what they generally mean.

Not owner: You are neither the user who mounted the repository on this machine nor root.

Device busy: Some process on your machine is cd'ed into a repository directory or has a file in the repository open. The lsof program is sometimes helpful in finding such processes, but in the worst case you may be reduced to rebooting to kill them all off.

See Also

vesta-intro(1), repository(8), mountrepos(8), vmount(8).
