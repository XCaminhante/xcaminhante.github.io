_is_bool(v: any): bool

Returns TRUE if v is a boolean value and FALSE otherwise. 

_is_bool(FALSE) == TRUE

_is_bool([ foo = 1 ]) == FALSE
_is_bool(<1, 2>) == FALSE
_is_bool("foo") == FALSE

_is_int(v: any): bool

Returns TRUE if v is an integer value and FALSE otherwise. 

_is_int(1) == TRUE
_is_int(0xabcde) == TRUE

_is_int(TRUE) == FALSE
_is_int(<1, 2>) == FALSE
_is_int("foo") == FALSE

_is_text(TRUE) == FALSE
_is_text([ foo = 1 ]) == FALSE
