Scoping : scoping1.ves

  • There are no global variables, but functions do capture their definition context:

    • {
    • x = 1;
    • f(y) { return x+y; };
    • x = 2;
    • return f(3);
    • }

  • The function body sees the first value for x, so the result is 4, not 5!
