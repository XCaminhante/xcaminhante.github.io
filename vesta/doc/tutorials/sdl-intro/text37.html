Binding Ops: bind_plus.ves

  • When used on bindings, + is called “binding overlay”:

    • {
    • b1 = [x=1, y=2];
    • b2 = [x=3, z=4];
    • return b1+b2;
    • }

  • Names in the right-hand operand take precedence, so this returns:

    • [ x=3, y=2, z=4 ]
