Binding Ops : bind_pp2.ves

  • ++ only recurses when the value on both sides are bindings

  • If only one is a a binding, the right-hand side value gets used (just like +):

    • {
    • b1 = [foo=1, bar/y=2];
    • b2 = [foo/u=3, bar=4];
    • return b1++b2;
    • }

  • In this case, the result is identical to b2
