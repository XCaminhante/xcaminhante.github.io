Scoping : scoping5.ves

  • Remember that all functions have an implicit final parameter “.” which is usually taken from the calling context, but can be passed explicitly:

    • {
    • f() { return ./x+1; };
    • . = [ x = 1 ];
    • return <f(), f([x = 3])>;
    • }

  • The result is:

    • <2, 4>
