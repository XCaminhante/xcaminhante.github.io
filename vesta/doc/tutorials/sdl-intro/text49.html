Scoping : scoping3.ves

  • A block of statements can be used as an expression, but assignments are local:

    • {
    • x = 1; y = 2;
    • z = { x += y; return x; };
    • return <x, y, z>;
    • }

  • x is unmodified by the block, so the result is:

    • <1, 2, 3>
