Scoping : scoping2.ves

  • If a function modifies a variable, that change is local:

    • {
    • x = 1;
    • f(y) { x += y; return x; };
    • return <f(2), f(3), x>;
    • }

  • x is unmodified by the function call, so the result is:

    • <3, 4, 1>
