Binding Ops : bind_pp.ves

  • Related to + is ++, the “recursive overlay” operator:

    • {
    • b1 = [foo/x=1, bar/y=2];
    • b2 = [foo/u=3, bar/v=4];
    • return <b1+b2, b1++b2>;
    • }

  • With +, names are replaced. With ++, nested bindings are recursively merged.

  • ++ is very useful for making directory structures
