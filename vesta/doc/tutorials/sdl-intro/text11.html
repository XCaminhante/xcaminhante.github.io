More Binding Syntax

  • Nested bindings made by specifying a path with names separated by slashes:

    • [ foo/a = 1, bar/b = 2 ]
    • [ foo = [ a = 1 ], bar = [ b = 2] ]

  • Placing a variable in binding with the same name as the variable:

    • [ foo, bar ]
    • [ foo = foo, bar = bar ]
