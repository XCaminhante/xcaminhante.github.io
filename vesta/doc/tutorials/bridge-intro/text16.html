Simple Bridge Details

  • ./build_root(<”grep”>)

    • Calls the function “./build_root” passing a list with a single element: the text string "grep"
    • ./build_root is a convention used by build environments to make it easier to construct a root filesystem out of several OS component packages
    • We just want the “grep” component we imported with pkg2vesta.pl, so that's all we ask for
